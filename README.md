# Installation and Usage

To view this project locally:

1.  **Clone the repository:**

    git clone https://github.com/mstanton6/307_SBA.git
    
2.  **Navigate to the project directory:**

    cd 307_SBA
    
3.  **Open the file:** Simply double-click the `index.html` file in your file explorer, or right-click and select "Open with Live Server" if you are using an extension like VS Code's Live Server.
